# Installation & Usage
[Back to top](#table-of-contents)

1. Download the pre-complied binary from the releases page.
2. Install it locally:
```
$ git clone https://github.com/Th3-S1lenc3/Matrix-Calculator.git
$ cd Matrix-Calculator
$ npm install
$ npm start
```
